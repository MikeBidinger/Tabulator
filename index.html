Filter Options:
  • Like: The "like" filter displays any rows with data that contains the specified string anywhere in the specified field (case insensitive)
  • Keywords: The "keywords" filter displays any rows with data containing any space separated words in the specified string (case insensitive)
  • Starts: The "starts" filter displays any rows with data that starts with the specified string (case insensitive)
  • Ends: The "ends" filter displays any rows with data that ends with the specified string (case insensitive)
  • RegEx: The "regex" displays rows that match a provided regular expression (case sensitive):
    • Brackets:
      • [abc]: Find any of the characters between the brackets
      • [0-9]: Find any of the digits between the brackets
      • (x|y): Find any of the alternatives separated with "|" (pipe)
    • Quantifiers:
      • n+: Matches any string that contains at least one n
      • n*: Matches any string that contains zero or more occurrences of n
      • n?: Matches any string that contains zero or one occurrences of n
